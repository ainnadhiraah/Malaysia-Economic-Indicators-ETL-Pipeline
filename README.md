# Malaysia-Economic-Indicators-ETL-Pipeline

Malaysia Economic Indicators ETL Pipeline (DOSM Open API) Built a Python-based ETL pipeline to ingest monthly and quarterly macroeconomic indicators (GDP, PPI, Wholesale &amp; Retail, Productivity) from DOSM Open APIs for downstream reporting and analysis.

Features
1) Fetches Haver data automatically from configured series codes or datasets
2) Converts raw Haver output into clean, tabular DataFrames
3) Handles date alignment and missing data gracefully
4) Saves extracted data into Excel format
5) Modular and reusable function structure (easy to add new datasets)
6) Compatible with other automation modules
